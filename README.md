# Triple

Create an Sqlite-backed Entity-Attribute-Value Datastore

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'triple'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install triple

## Usage

Before you can use Triple, you must require it in your application:

```ruby
require 'triple'
```

```
db = Triple.new('...')
```
